Car Judge

Link to our github repo: https://github.com/varickt/Group1-CapstoneProject2024

As part of my group's final capstone project, we built Car Judge, a web application that displays detailed car information including make, model, year, price, and reviews. The project was developed from scratch using technologies like Prisma, React, PostgreSQL, and more.

My Role:
I focused primarily on the backend development:

- Developed the server using Node.js, PostgreSQL, and Nodemon.
- Created and managed multiple routes, including authentication via JWT tokens.
- Implemented data seeding using Prisma, TablePlus, FakerJS, and a curated set of car images to match the car models.
- Secured user registration with bcrypt to hash passwords.

One of my favorite challenges was integrating MockEmailService for the "forgot password" feature on the login page. I also contributed to the React frontend, collaborating closely with the team to ensure a smooth, functional, and visually appealing experience.

Outcome:
Through strong teamwork and collaboration, we built a fully functional website that not only works seamlessly but also looks great.
