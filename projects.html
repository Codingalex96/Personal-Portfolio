Car Judge

GitHub Repo: Car Judge

As part of my group's final capstone project, we built Car Judge, a web application that displays detailed car information including make, model, year, price, and reviews. The project was developed using Prisma, React, PostgreSQL, and more.

My Role:
- Developed the backend with Node.js, PostgreSQL, and JWT authentication.
- Created data seeding using Prisma, TablePlus, and FakerJS.
- Implemented bcrypt password hashing and a mock email service for "forgot password."

Outcome: Built a fully functional and visually appealing web app through teamwork.
